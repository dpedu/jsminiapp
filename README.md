# jsminiapp

A barebones framework for quick & tiny javascript/python apps.

## Requirements

* Python 3.2
* CherryPy3
* Jinja2

## Included Libraries:

* Handlebars for client-side templates
* Bootstrap 3 theme
* Handlebar Helpers: https://github.com/danharper/Handlebars-Helpers
